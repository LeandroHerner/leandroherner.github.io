Generación Aurora

Generación Aurora es una solución de Business Intelligence basada en un caso real desarrollado para Mondelez International (datos completamente anonimizados). Se trata de una herramienta de gestión interna para los Responsables Comerciales (RCs), diseñada para monitorear un programa de incentivos orientado a potenciar la red de distribución.

El valor central del dashboard es permitir a cada RC conectar el desempeño operativo de sus cuentas con el resultado competitivo global. El reporte facilita un seguimiento dual: visualizar el Ranking Nacional para entender la posición de sus clientes frente al resto, y auditar en detalle los KPIs específicos (ventas, objetivos, mix) que cada distribuidor debe cumplir para escalar posiciones.

A nivel técnico, el proyecto implementa una arquitectura de seguridad basada en Row-Level Security (RLS) adaptada al organigrama comercial. Esto garantiza una experiencia eficiente: mientras el Ranking es visible para todo el equipo comercial, la Base Plana aplica filtros dinámicos para que cada RC acceda exclusivamente al detalle transaccional de su cartera de clientes asignada.
