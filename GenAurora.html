Generación Aurora

Generación Aurora es una solución de Business Intelligence creada a partir de un caso real desarrollado para Mondelez International (con datos completamente anonimizados). Es una herramienta interna pensada para los Responsables Comerciales (RCs), que les permite monitorear un programa de incentivos y entender cómo se mueve su red de distribución.

El foco principal del dashboard es ayudar a cada RC a conectar el desempeño operativo de sus cuentas con el resultado competitivo general. El reporte ofrece un seguimiento dual: por un lado, visualizar el Ranking Nacional para conocer la posición de sus clientes frente al resto; y por otro, auditar en detalle los KPIs clave —ventas, objetivos, mix— que cada distribuidor debe cumplir para escalar posiciones dentro del programa.

A nivel técnico, el proyecto implementa una arquitectura de seguridad basada en Row-Level Security (RLS), alineada con la estructura comercial. Esta configuración asegura una experiencia fluida y segura: el Ranking es accesible para todo el equipo, mientras que la Base Plana aplica filtros dinámicos que permiten a cada RC ver únicamente el detalle transaccional correspondiente a su cartera de clientes.
